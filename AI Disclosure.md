# AI Use Disclosure

As required by the submission guidelines, this outlines our use of AI coding assistants.

## How We Used AI
We used Claude to help draft boilerplate code, debug errors, clean up our code, and speed up repetitive tasks like plotting and writing tests. We thoroughly reviewed, tested, and manually edited all AI-generated code before including it in our project.

## What We Did Ourselves
All core modeling decisions, methodology, result interpretations, and written summaries are entirely our own work. We did not use AI to write our analysis or draw conclusions.

## Code Tracking
Any code drafted with AI help is clearly marked with an `# [AI-ASSISTED]` comment at the top of the cell or block, making it completely traceable throughout our notebooks.
